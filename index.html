HTML CODE NOTES

Part 1: The Basics

- Always start a html file with "!DOCTYPE html"

- Always put a "html" to indicate the code has started

- Visible part of an html file is between the "body"

- "h" stands for header. There are 6 different sizes, with 1 being largest and 6 being smallest.
It is written like this: "hx", x standing for an integer between 1 and 6

- "p" stands for paragraph. This is text at a regular size

- This is a link, using "a"

- Images are inputted using "img"

Part 2: The Elements

- An element is essentially a line in code

- There are also nested elements, such as "html", "body", that contain others

- "br" means break line, resulting in a space, similar to what \n would do in code

Part 3: Attributes

- When using "a", you use "href=" to tell the program where to find the link at

- When using "img", you add "src=" to tell the program where to find the image:
There are two types of src's:
1. Absolute URL, meaning the photos are hosted on another website (may infringe copyright)
2. Relative URL, meaning photos are within your website

- The "img" tag should also include the width and height of the image, being "width=""" and "height=""""

- "alt" is text put after a "img" in case the image cannot be rendered

- "lang" is put right after "html", used to specify the language of the page: "html lang="""
By also adding a "-xx" to "lang", you can also add the country of the code
